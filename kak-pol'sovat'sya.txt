1. Есть 3 компонента:
    - insta-grid(это собственно то чтоо посередине) - твоя сетка инсты
    - pre-viewer(это то что слева) - где хранятся заготовки, там 5 папок(деление на плвны: крупный/средний/дальний/портреты/абстрактное)
    - referencer(это то что справа) - туда пихаются скрины чужих инста-сеток(оптмальное - по 4 ряда) на визуал которых мы опираемся когда делаем свою
2. Есть модалка, но она недоделанна lol, можно просто нажать на фотку и посмотреть как будет если открыть одну, можно посмотреть на черном/белом фоне(кнопка солнца/луны)
3. Есть localstorage который вообще никакого к этому отношения не имеет lol
    в нем хранятся фоточки(ага, локально и надо все вручную вписывать через пути)
    как делится все там:
    в src есть папка img, там собственно храним все картинки
    в ней 3 папки(названия соответствуют компонентам выше)
    в папке previewer 5 папок тоже названия по планам крупности/приинадлежности
    soooo, пихаем в каждую папочку все что надо, меняем названия фоток на порядковые номера потом новые пиихать(так удобнее имхо)
    дальше все функционирует на айдишниках:
    id (1-12) - pre-viewer - крупный план
    id (13-36) - pre-viewer - средний план(тут побольше потому что к меня их обычно больше всех)
    id (37-48) - pre-viewer - дальний план
    id (49-60) - pre-viewer - портреты
    id (61-72) - pre-viewer - абстрактное
    id (100+) - insta-grid - сетка инсты(поставила ограничение до 500)
    в pre-viewer можно добавить id до 100, но перепсывать (id) надо все вручную

    сами объекты фото состоят из id, src, w, h, text, tags, где:
    - w - ширина (если картинка горизонтальная пишем auto, если вертикальная оставим пустые скобки)
    - h - высота (аналогично w, но пишем inherit)
    text, tags - когда-нибудь доделаю lol
    
    в инста-сетке первые три идут закрепом(100, 101, 102), пустое окно для добавленя новой фотки на 103 id, поменять конечно можно но тогда надоо весь функционал править
    следовательно когда добавляем новую фотку(допустим уже то сто выложили у себя и то что точно там будет всегда),
    она должна быть на 104 id(первые три закрепа + пустая на 103 для новой)

4. Есть стрелки туда-сюда рядом с инста-сеткой, они добавляю еще пустых ячеек либо убирают (убтрают вместе с фотками так что аккуратнее, потом это доработаю lol)

5. drag-n-drop работает на самой сетке инсты, в previewer и между этими двумя компонентами
6. Фотки для рефов(их src) вписываем в сам компонент referencer а не в localstorage
    там можно увеличить кол-во (все скроллится как, кстати, и сетка инсты)
7. снизу сетки инсты тоже кнопочка солнца/луны - это все для того чтобы понмать как будет выглядеть
    если у человека светлая/темная тема на телефоне(это оказалось важно для меня лично)
8. боковые окна можно закрывать/открывать(есть стрелочки) чтоб сфокусироваться на виде сетки без отвлекающих картинок

9. ну и есть компонент planner, в котором описывается большая часть функционала
